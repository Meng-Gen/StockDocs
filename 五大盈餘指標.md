# 五大盈餘指標
## 1. 存貨指標 (-)
- 存貨指標 = 存貨增加率 - 銷貨收入增加率
- 存貨增加率 = (本期存貨 - 前兩期存貨平均) / 前兩期存貨平均
- 銷貨收入增加率 = (本期銷貨收入 - 前兩期銷貨收入平均) / 前兩期銷貨收入平均
